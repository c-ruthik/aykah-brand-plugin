# Certifications Glossary

Single source of truth for how to cite the certifications Aykah products carry. Prevents the "AI invents a fake certification" failure mode.

**Hard rule:** only cite a certification if the user has confirmed THIS product carries it. Never cite from memory of past products. Ask if uncertain.

---

## CertiPUR-US

- **What it certifies:** flexible polyurethane foam — tested for harmful chemicals, emissions, and durability.
- **Citation format:** `CertiPUR-US®` (with the registered-mark symbol on first mention; subsequent mentions can drop it).
- **Premium phrasing:** "High-density foam, CertiPUR-US® certified — tested for harmful chemicals, emissions, and durability."
- **Don't say:** "non-toxic foam", "safe foam", "eco-foam" (vague — name the specific certification).
- **Where to verify:** certipur.us

## OEKO-TEX STANDARD 100

- **What it certifies:** textiles screened against 1,000+ harmful substances.
- **Citation format:** `OEKO-TEX® STANDARD 100` on first mention. If a certificate ID is available, cite it: `OEKO-TEX® STANDARD 100, certificate XX.HUS.XXXXX, [Institute name]`.
- **Premium phrasing:** "Belgian linen, OEKO-TEX® STANDARD 100 certified — screened against 1,000+ harmful substances."
- **Don't say:** "chemical-free fabric", "safe textile", "non-toxic linen" (vague — name the specific standard).
- **Where to verify:** oeko-tex.com

## FSC (Forest Stewardship Council)

- **What it certifies:** wood from responsibly managed forests.
- **Citation format:** `FSC®-certified` or `FSC® certified` on first mention. Mentioning the chain-of-custody designation (FSC 100%, FSC Mix) is more precise but only if confirmed.
- **Premium phrasing:** "Frame: FSC®-certified hardwood — sourced from responsibly managed forests."
- **Don't say:** "sustainable wood", "eco-wood", "responsibly sourced timber" (vague — name FSC explicitly).
- **Where to verify:** fsc.org

## GREENGUARD Gold

- **What it certifies:** strict chemical emission limits — typically applied to upholstery, finishes, furniture.
- **Citation format:** `GREENGUARD Gold` (one word for GREENGUARD, capitalized). The base GREENGUARD certification exists separately; "Gold" is the stricter tier.
- **Premium phrasing:** "Finish: GREENGUARD Gold — meets strict chemical emission limits suitable for sensitive environments like schools and healthcare."
- **Don't say:** "low-VOC", "low-emission" without the certification name (vague).
- **Where to verify:** spot.ul.com (UL Solutions hosts GREENGUARD)
- **Critical:** if a product is GREENGUARD-certified but NOT Gold, do NOT write "GREENGUARD Gold". Use "GREENGUARD certified" only.

## Citation order in PDP specs

When listing multiple certifications:

1. Frame / wood — FSC
2. Foam / cushion — CertiPUR-US
3. Textile / upholstery — OEKO-TEX
4. Finish / VOC — GREENGUARD Gold

Group related materials with their certifications. Don't list certifications in a separate "Certifications" block divorced from the material — readers parse certificate-as-property-of-material more easily.

---

## What NOT to do

- **Don't claim certifications the product doesn't actually carry.** Verify first. If uncertain, ask the user.
- **Don't generalize ("eco-friendly", "non-toxic", "safe").** Name the specific certification or remove the claim.
- **Don't drop the registered-mark symbol on first mention** in formal copy (PDP, About, packaging). It's required for legal use of these marks.
- **Don't claim certifications the user hasn't confirmed for THIS specific product.** A product line may have changed suppliers.
- **Don't invent certificate ID numbers.** If you don't have the ID, cite the standard without it.

---

## When facts are uncertain

If a user provides material info but doesn't mention certifications, ASK before drafting:

> "Does this product carry any of these — CertiPUR-US foam? OEKO-TEX textile? FSC wood? GREENGUARD Gold finish? If yes, certificate IDs help. If unsure, say 'unknown' and I'll write around them."

Don't infer certifications from material descriptions ("Belgian linen → OEKO-TEX" is NOT a safe inference).

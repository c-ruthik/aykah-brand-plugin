01 · Brand Foundation
Four Brand Values
Every color and font decision must serve these four values. If a design choice doesn't map to at least one, it doesn't belong.
ValueWhat It MeansHow Colors Serve ItHow Fonts Serve It
Comfort Warmth, softness, home. Customers should feel like they're already in a cozy room. Warm golds, creamy ivories, soft backgrounds. The palette avoids cold blues/greys. Libre Baskerville's transitional serifs convey heritage and authority. Manrope's rounded terminals feel warm and modern.
Integrity Honest craftsmanship. No gimmicks, no fake luxury. Indigo navy (#363B57) — deep, confident, no-nonsense. Not flashy black, not trendy teal. Serif headings carry centuries of typographic heritage. Honest, established, proven.
Friendly & Approachable Not intimidating. Customers feel welcome, not judged by their budget. 60% warm whites dominate the page. Gold accents are inviting, not exclusive. Manrope's geometric-humanist forms balance friendliness with premium clarity.
Trustworthy & Reliable Customer trusts Aykah enough to spend $500-$900 online without touching the product. Navy = reliability. Green for verification. Consistent color usage builds predictability. Serif fonts increase perceived trustworthiness by ~40%. Clear body text = nothing hidden.

03 · Color Rules — Critical
Do's and Don'ts

Accent Color — Three Tokens, Three Roles

Do: Use --accent (#B8956A) for decorative elements only: star fill icons, decorative lines, background gradients, icon fills, hover glows.
Do: Use --accent-text (#8B6E4A) for any gold-colored text: links, labels, eyebrow categories, price highlights. This passes 4.5:1 on white.
Do: Use --accent-btn (#7D6040) for buttons with white text. This passes 4.5:1 for both text and UI contrast.
Never: Use --accent (#B8956A) as text color. It fails WCAG at 2.78:1. Zero exceptions.
Never: Put white text on --accent (#B8956A) background. It fails at 2.78:1. Use --accent-btn (#7D6040) instead.

Text Colors

Do: Use --text-body (#2C2C2C) for all body text, product descriptions, specs.
Do: Use --text-muted (#6B6560) for secondary info: delivery times, timestamps, breadcrumbs.
Do: Use --text-light (#76716B) for helper text, captions, minor labels.
Never: Use --text-disabled (#B8B2A8) for any readable content. It is exclusively for disabled/inactive UI states.

Borders

Rule: --border (#E8E3DB) at 1.28:1 fails as a standalone UI boundary. Always pair it with box-shadow or a background color difference.

Background Ratio — 60 / 30 / 10

Do: 60% warm whites (--white, --ivory, --warm) · 30% indigo (--primary, --primary-dark) · 10% gold + green accents.
04 · Typography
Two Fonts. No Exceptions.
Heading font: Libre Baskerville. Body font: Manrope. Never add a third font.

Typography Rules

Do: Eyebrows always: UPPERCASE + letter-spacing: 3px + font-weight: 600 + color: --accent-text.
Do: Buttons always: UPPERCASE + letter-spacing: 1.5px + font-weight: 600.
Do: Prices always in Libre Baskerville (serif numerals feel more premium).
Do: Testimonials always in Libre Baskerville italic 400.
Never: Use Libre Baskerville for body text. It is headings and display only.
Never: Use Manrope for product names or prices.
Never: Use a third font anywhere.
Never: Go below 10px for any text.

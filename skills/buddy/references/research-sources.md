# Research Sources ‚Äî for the aykah-researcher agent

When buddy needs current trend data, competitor moves, or external signals, it dispatches the `aykah-researcher` agent. That agent uses this list to decide where to search.

If the user names a specific site, the agent searches **only** that site. Otherwise, it picks 2‚Äì4 sources from the list below based on the topic.

## Curated source list

### Reddit (community signal)

Use for: real consumer language, complaints, what people actually buy, discussion of trends.

- `r/InteriorDesign` ‚Äî design discussion, frequent professional voices
- `r/HomeDecorating` ‚Äî broader consumer audience
- `r/Furniture` ‚Äî buying advice, brand comparisons (Aykah, Structube, Article often discussed)
- `r/SmallSpace` ‚Äî condo/apartment buyers, core Aykah audience
- `r/BuyItForLife` ‚Äî durability + investment-piece buyers
- `r/AmIBeingUnreasonable`, `r/MaliciousCompliance` ‚Äî for delivery / customer-service incident research
- `r/TheGirlSurvivalGuide`, `r/Adulting` ‚Äî first-furniture buyers
- `r/Calgary`, `r/vancouver`, `r/toronto` ‚Äî local market signals

## Source-selection heuristic

| Topic | First 2‚Äì3 sources to check |
|---|---|
| Furniture trend / aesthetic | Pinterest Predicts, Dwell, Apartment Therapy |
| Consumer perception of a brand | Reddit r/Furniture, r/HomeDecorating |
| Campaign / ad creative benchmarks | AdAge, AdWeek, the competitor's Instagram |
| Pricing positioning | Direct competitor sites |
| Material spec / certification | The certifying body's own site |
| Customer-service complaint patterns | Reddit (search the brand name) |
| Search-demand validation | Google Trends + Exploding Topics |
| Local market signal (Calgary, Toronto, Vancouver) | r/<city> + local design publications |

---

## When the user names a site

If the user says "search Reddit's r/InteriorDesign", "check Dezeen", or any specific site/subreddit, the researcher restricts scope to ONLY that source. Do not pad with other sources unless explicitly asked.

---

## What NOT to use

- AI-generated content farms (low-quality summaries)
- Aggregator sites without primary reporting (most "Top 10" listicles)
- Stock-photo sites (visual reference is fine, but not for trend analysis)
- Older than 18 months for trend topics; older than 5 years for material/certification facts (those are stable)

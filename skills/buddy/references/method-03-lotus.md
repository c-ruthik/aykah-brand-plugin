# Method 3 — Lotus Blossom

**Source:** Yasuo Matsumura, Clover Management Research. Used by the Japan Management Consultants Association.

---

## What it does

Generates 64 structured ideas through systematic expansion: **1 central theme → 8 sub-themes → 64 specific ideas.**

## When to use

When you need *coverage*, not just a few good ideas:

- Content strategy across channels
- Product line expansion
- Loyalty program design
- Programmatic SEO topic maps
- Brand-positioning exploration
- Annual marketing calendar

Especially when the failure mode would be "pursuing the obvious idea while missing a category of approaches entirely."

## Workflow

### Step 1 — Lock in the central theme

Specific and singular. Vague themes produce vague petals.

- **Bad:** "Aykah marketing"
- **Good:** "How to make The A-List feel exclusive without being expensive to run"

### Step 2 — Generate 8 sub-themes (the inner ring)

These are *categories* of ideas, not specific ideas. Pick a slicing axis and stick to it:

| Axis | Example sub-themes (pick 8) |
|---|---|
| By stakeholder | customer / employee / vendor / designer / influencer / repeat buyer / first-time / lapsed |
| By stage | awareness / interest / consideration / purchase / delivery / ownership / repeat / advocacy |
| By channel | in-store / web / email / Instagram / TikTok / paid / partnership / direct outreach |
| By benefit type | access / status / savings / content / community / education / recognition / surprise |
| By format | event / content / product / service / tool / partnership / system / ritual |
| By time horizon | today / week / month / quarter / year / next year / 3-year / 5-year |

Don't mix axes — that creates overlapping themes.

### Step 3 — For each sub-theme, generate 8 specific ideas

Concrete and specific.

- **Bad:** "improve customer relationships"
- **Good:** "Send a handwritten note from Salman to top-50 customers in November"

Total: 64 ideas.

### Step 4 — Surface patterns

This is where Lotus Blossom produces insight. The 64 ideas are inputs to insight, not the insight itself.

- Which sub-themes produced the strongest ideas? (concentration = signal)
- Which were hardest to fill? (gaps = either irrelevant OR underexplored)
- Which ideas across sub-themes combine into bigger ideas?
- Which 5–10 are worth carrying forward?

## Output format

```
Central theme: <one specific sentence>
Slicing axis: <stakeholder / stage / channel / benefit / format / time>

Sub-theme 1 — <name>
  1.1 <idea>     [Maturity: Raw]
  1.2 ...
  ...
  1.8 ...

Sub-theme 2 — <name>
  ...

[continues for all 8]

Patterns:
  - Concentration: sub-themes <n, n> produced the strongest ideas because ...
  - Gaps: sub-themes <n, n> were hard to fill because ...
  - Cross-cutting: ideas X.Y + W.Z combine into ...

Carry-forward (5–10): <list with maturity tags>
Watch out for: <banned words / clichés to scrub before delivery>
```

After Lotus, route to **Method 8 — Converge** to narrow the carry-forward to 1–3 winners.

## Common failure modes

- Vague central theme produces vague petals
- Overlapping sub-themes (slicing axis was wrong — restart)
- Stopping at 30 ideas because it feels complete (force to 64)
- Skipping the pattern-surfacing step
- Treating each idea as equally serious (most are junk; that's expected)

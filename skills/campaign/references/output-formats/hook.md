# Output format — `hook`

Used when the user asks for a single ad hook, post hook, scroll-stopper, or opening line.

## Methods invoked
- `08-five-stages.md` (hooks are stage-specific — primary lens)
- `07-two-tone.md` (functional + emotional pairing inside the hook)
- (Plus grounding: JTBD, ICP, Dunford, 5 Pillars)

## Workflow inside the skill

1. Confirm what surface the hook is for (Meta cold ad, retargeting, IG carousel opener, Reels opening, blog headline, etc.)
2. Identify the audience awareness stage
3. Generate 5 candidates spanning angles:
   - Problem-aware (Stage 2): name the pain
   - Solution-aware (Stage 3): comparison frame
   - Product-aware (Stage 4): social proof / specifics
   - Most aware (Stage 5): direct, factual
   - Anxiety-reducing (any stage): address the dominant Force for Aykah
4. Voice-gate all 5
5. Drop any below pillar-fit 3
6. Recommend one

## Hook quality bar

- **6–14 words preferred** (varies by surface — IG carousel allows longer, TikTok wants shorter)
- Holds two tones (functional + emotional) where possible
- Names a concrete material, fact, location, OR sensory moment
- No exclamation marks
- No urgency phrases
- No "elevate", "transform", "unlock", "curated experience"
- No "are you tired of…" or "imagine if…" openers (banned AI-tells)
- Should sound like a sentence Sanya would say, not a slogan
- Stage-1 hooks are NOT recommended for Aykah currently — too far up funnel for our budget. Default to Stage 2–4.

## Deliverable shape

```
TYPE: hook
MODE: <quick / standard / deep>
CONTEXT: <one-line restatement; include surface — Meta cold ad / retargeting / Reels / etc.>

AUDIENCE STAGE: <2 / 3 / 4 / 5>  (with reasoning)

CULTURAL MOMENT (standard + deep only):
<one paragraph from researcher agent>

────────────────────────────────────────────────
HOOK CANDIDATES (5)

1. <Hook line>
   Stage targeted: <2/3/4/5>
   Angle: <problem-aware | comparison | social-proof | direct | anxiety-reducing>
   Functional anchor: <material / fact / location>
   Emotional anchor: <feeling / signal>
   Force served: <Push / Pull / Anxiety / Habit>
   Pillar-fit: <1–5>  (deep mode only)

2. ...

(continues to 5)
────────────────────────────────────────────────
RECOMMENDED PICK: #<N>
Why: <2–3 sentences — stage match + Force + ICP fit>

PAIRED BODY DIRECTION:
<one sentence on the body the hook should set up — material focus / customer photo / showroom shot / etc.>
PAIRED CTA DIRECTION:
<one sentence on the matching CTA — book showroom / request swatch / shop PDP>
────────────────────────────────────────────────

VOICE-GATE: APPROVED | BLOCKED-AND-FIXED
```

## Quick mode shortcut

Return 5 hooks only, no anchor/stage/Force breakdown, no recommendation, no body/CTA pairing. Voice gate skipped.

## Common failure modes for this type

- Hooks targeting Pull when the audience's blocker is Anxiety
- Hooks with banned openers ("are you tired of…", "imagine if…", "in today's world…")
- Hooks too long to land in 3 seconds (over 15 words on Meta cold)
- Hooks that work at Stage 5 but get used in Stage 1–2 cold (Stage-5 facts feel cold to cold traffic)
- Hooks that read like every other DTC furniture brand (no Aykah specificity)
- Hooks without a concrete noun (oak, bouclé, walnut, Burnaby, $499) — too abstract to convert

# Pinterest

Pinterest is **search**, not social. Treat it like SEO with images.

Furniture has unusually long Pinterest shelf-life — pins drive saves and clicks for months, not days. Sundays Furniture has shifted budget toward Pinterest specifically for this reason.

---

## Channel dial

Voice Guide v2 doesn't list Pinterest in its Channel Guide explicitly. Default it to: **Medium warmth, Medium formality, Materials hook + Living hook, keyword-led.**

The dial is a step more functional than IG/Reels because Pinterest users are *planning* — they're saving for a kitchen renovation, a move, a holiday refresh. Less scene-hook, more usefulness.

---

## Pin description

The pin description is the most important thing you'll write for Pinterest. It's both UX (read by users) and SEO (read by Pinterest's algorithm).

### Length

- **500 characters max.** Use 350–500 — Pinterest weights longer descriptions higher.
- **First sentence carries the keyword.** Pinterest truncates in feed; the first ~80 chars are what users see.
- **No hashtags.** Pinterest deprecated hashtags in 2020; they no longer help discovery.

### Structure

```
[Keyword phrase, frontloaded] – [product name] in [material/color]. [One sentence on craft or use-case]. [One sentence on how it lives in a room]. [One sentence on practical detail — dimensions, care, or where to see it]. [Soft CTA].
```

### Keyword targeting

Pinterest is a search engine. Lead with the search query a real person would type:

**Aykah-relevant Pinterest queries to target (research these in Pinterest Trends):**

- "modern Canadian furniture"
- "small condo living room ideas"
- "Belgian linen sofa"
- "solid oak dining table"
- "modern apartment storage bed"
- "neutral living room sectional"
- "kiln-dried oak furniture"
- "modern furniture Vancouver / Toronto"

The first sentence of the pin description should contain the primary keyword phrase verbatim, not a synonym.

---

## Pin title

- **100 characters max.** Use 60–80.
- **Keyword frontloaded.** Same query as the description's lead.
- Title format: `[Keyword phrase] | [Product name] | Aykah`

Example: `Modern Belgian Linen Sectional | Aura Sofa | Aykah`

---

## Image requirements

- **Aspect ratio:** 2:3 (vertical) or 1:1 (square). 2:3 wins on engagement.
- **Resolution:** 1000 × 1500 px minimum.
- **No heavy text overlay.** A short title across the top in Manrope is fine. Don't cover the product.
- **Branded subtly.** Aykah wordmark in a corner; not center-stage.

---

## Boards

- Organize boards by **use case or room**, not by product type. Users search "small living room" not "sectionals".
- Board titles should be 2–4 words, keyword-led.
- **Recommended boards for Aykah:**
  - Modern Living Rooms
  - Small-Space Furniture
  - Belgian Linen & Natural Materials
  - Solid Wood Dining
  - Bedroom Storage Ideas
  - Canadian Modern Home
  - Showroom Looks (Vancouver / Toronto)

---

## What does NOT belong on Pinterest

- Hashtags
- Urgency / discount language ("limited time," "% off")
- Founder-voice scene hooks ("Sunday morning, second cup") — save those for IG
- Reels-style ASMR captions — Pinterest is read, not heard
- Banned words list applies in full

---

## Output format for Pinterest

```
PIN TITLE:
<60–80 chars, keyword-led>

PIN DESCRIPTION:
<350–500 chars, keyword frontloaded, no hashtags>

ALT-TEXT:
<descriptive, screen-reader friendly>

BOARD:
<which existing board this fits, or new board name>

IMAGE NOTE:
<aspect ratio + key visual + any text overlay>

VOICE-GATE: APPROVED
PERSONAS: 3/3 passed
```

## Examples

### PASS

**Pin title:**
Modern Belgian Linen Sectional | Aura Sofa | Aykah

**Pin description:**
Belgian linen sectional for modern living rooms — the Aura Sofa in oat colorway, kiln-dried oak frame, 280 GSM performance weave. Designed for small condos and open-plan apartments where the sofa anchors the room. Sink-in support, soft-close storage drawers under each cushion. Available in Burnaby, Port Coquitlam, and Mississauga showrooms. Explore the collection at aykah.ca.

**Alt-text:**
Aura Sectional in oat Belgian linen, three-quarter view in a modern living room with afternoon light.

**Board:** Modern Living Rooms

---

### FAIL — same product, off-brand for Pinterest

**Pin title:**
Stunning Luxury Sectional - LIMITED TIME!

**Pin description:**
Don't miss out on the most beautiful sectional you'll ever own! Sunday morning, second cup, this is what we live for. Free shipping over $199 — get yours before they're gone! #FurnitureSale #LuxuryLiving #HomeInspo

**Why it fails:**
- Banned: "Luxury", "LIMITED TIME", "Don't miss out", "free shipping", urgency
- Hashtags don't belong on Pinterest
- Caps lock + exclamation marks
- "Sunday morning, second cup" is an IG hook, wrong dial for Pinterest's planner audience
- Title doesn't lead with a real search keyword
- Description is emotional, not functional — Pinterest users want utility
